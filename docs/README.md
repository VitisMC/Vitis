# Vitis Documentation

Documentation is available in two languages:

- **[English](eng/)** — Full documentation in English
- **[Русский](rus/)** — Полная документация на русском языке

## Structure

```
docs/
├── eng/
│   ├── architecture.md    — System architecture overview
│   ├── protocol.md        — Protocol layer overview
│   ├── roadmap.md         — Development roadmap
│   ├── generators/
│   │   ├── README.md          — Data generators overview (59 generators)
│   │   ├── data-sources.md    — External data sources
│   │   └── version-upgrade.md — Version upgrade guide
│   ├── packets/
│   │   ├── packet-ids.md          — Packet ID system
│   │   └── updating-packet-ids.md — Updating packet IDs for new versions
│   └── registry/
│       ├── overview.md        — Registry system architecture
│       └── upgrade-guide.md   — Registry upgrade step-by-step
└── rus/
    ├── architecture.md    — Обзор архитектуры системы
    ├── protocol.md        — Обзор протокольного уровня
    ├── roadmap.md         — Дорожная карта развития
    ├── generators/
    │   ├── README.md          — Обзор генераторов данных (59 генераторов)
    │   ├── data-sources.md    — Внешние источники данных
    │   └── version-upgrade.md — Руководство по обновлению версии
    ├── packets/
    │   ├── packet-ids.md          — Система Packet ID
    │   └── updating-packet-ids.md — Обновление Packet ID для новых версий
    └── registry/
        ├── overview.md        — Архитектура системы регистров
        └── upgrade-guide.md   — Пошаговое обновление регистров
```
