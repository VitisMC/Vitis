Высокопроизводительное серверное ядро Minecraft, написанное на Go. В настоящий момент поддерживает **Minecraft 1.21.4** (протокол 769).

> **Статус: Экспериментальный / В разработке**
> Vitis находится в активной разработке. Пока не подходит для использования в продакшене.

## Возможности

### Сеть и протокол
- Полная реализация протокола Minecraft 1.21.4 (handshake, login, configuration, play)
- Онлайн-аутентификация (серверы сессий Mojang, RSA-шифрование)
- Сжатие пакетов (zlib)
- Горячая перезагрузка конфигурации

### Мир
- Чтение/запись файлов регионов Anvil (.mca)
- Генерация ландшафта на основе шума (simplex-холмы, вода, пещеры)
- Генерация плоского мира
- Асинхронная загрузка чанков с ограниченным пулом воркеров
- Стриминг чанков с индивидуальной дистанцией обзора и очередью приоритетов
- Граница мира с плавными переходами
- Цикл дня/ночи (тикание мирового времени)

### Геймплей
- Движение игрока, синхронизация позиции и телепортация
- Установка и разрушение блоков с сохранением мира
- Творческий и режим выживания
- Урон от падения, голод, естественная регенерация, истощение
- Смерть, респаун и бой (PvP с отбрасыванием, кулдаун атаки, урон от оружия, широкий удар, криты)
- Управление инвентарём (клик, shift-клик, перетаскивание, цифровые клавиши)
- Система крафта (1284 рецепта, сетки 2×2/3×3 с зеркальным отражением)
- Поедание еды с реестром 40+ предметов и восстановлением голода/насыщения
- Скорость копки с уровнями инструментов, правилами добычи и дропом блоков
- Атрибуты сущностей с модификаторами и отслеживанием изменений
- Система брони и экипировки с ванильным снижением урона
- Сохранение данных игрока (позиция, режим игры, здоровье, инвентарь)

### Сущности
- Физика сущностей с AABB-коллизиями
- Предметы с подбором, объединением и исчезновением
- Сферы опыта с притяжением к игроку
- ТНТ с обратным отсчётом и ray-cast взрывом
- Снаряды (стрелы, снежки, яйца, жемчуг Края)
- Транспорт (лодки, вагонетки)
- Падающие блоки

### Мультиплеер
- Синхронизация и видимость сущностей (анимация, приседание, спринт, экипировка)
- Скорборд и команды
- Боссбары
- Таб-лист с заголовком/подвалом
- Чат (замаскированные сообщения)
- Автодополнение команд
- Трансляция звуков и частиц
- Система прав операторов

### Инфраструктура
- 59 генераторов кода для данных протокола (блоки, предметы, сущности, звуки и т.д.)
- Полная система реестров (78+ реестров из ванильных данных)
- Структурированное логирование с настраиваемыми уровнями
- Тик-луп с настраиваемым TPS и обработкой перегрузки
- Шина событий с приоритетной сортировкой
- Система команд с автодополнением

## Быстрый старт

### Требования

- **Go 1.23** или новее
- **Git**

### Сборка

```bash
git clone https://github.com/vitismc/vitis.git
cd vitis
go build -o vitis ./cmd/vitis
```

### Запуск

```bash
# Скопируйте пример конфигурации
cp configs/vitis.yaml vitis.yaml

# Запустите сервер
./vitis -config vitis.yaml
```

Сервер по умолчанию слушает на `0.0.0.0:25565`. Подключайтесь клиентом Minecraft 1.21.4.

### Конфигурация

Смотрите [`configs/vitis.yaml`](configs/vitis.yaml) — все доступные параметры, включая:
- Настройки сервера (хост, порт, макс. игроков, онлайн-режим, MOTD)
- Настройки сети (таймауты, сжатие, размеры буферов)
- Настройки мира (дистанция обзора, воркеры чанков, TTL выгрузки)
- Настройки тиков (целевой TPS, поведение при отставании)
- Логирование (уровень, формат)

## Архитектура

```
cmd/vitis/          Точка входа
internal/
  network/          TCP-листенер, управление соединениями, пулы воркеров
  protocol/         Кодек пакетов, VarInt, буфер, конечный автомат
  session/          Жизненный цикл сессии, маршрутизация пакетов, обработчики
  registry/         Система реестров Minecraft (78+ реестров)
  world/            Менеджер мира, система чанков, генерация ландшафта
    chunk/          Загрузка чанков, хранение, асинхронные воркеры
    streaming/      Стриминг чанков для каждого игрока с очередью приоритетов
    terrain/        Генераторы: шумовой и плоский
    region/         Формат файлов Anvil .mca
    level/          level.dat и сохранение данных игроков
    persistence/    Абстракция хранилища чанков
  entity/           Типы сущностей, физика, метаданные
    physics/        AABB-коллизии, гравитация, движение
    projectile/     Стрелы, бросаемые предметы
    vehicle/        Лодки, вагонетки
  block/            Реестр блоков, поведения, жидкости
  item/             Реестр предметов
  inventory/        Контейнеры и управление окнами
  command/          Реестр команд и встроенные команды
  config/           YAML-конфигурация с горячей перезагрузкой
  logger/           Структурированное логирование (log/slog)
  event/            Шина событий
  tick/             Тик-луп и планировщик
  data/
    generator/      59 генераторов кода
    generated/      Сгенерированный Go-код (НЕ РЕДАКТИРОВАТЬ)
  scoreboard/       Скорборд и команды
  bossbar/          Боссбары
  chat/             Кодирование текстовых компонентов
  nbt/              NBT-кодировщик/декодировщик
  auth/             Аутентификация Mojang
  operator/         Права операторов
configs/            Примеры файлов конфигурации
docs/               Документация (английский и русский)
scripts/            Скрипты сборки и загрузки данных
```
